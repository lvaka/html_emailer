HTML Emailer Service

This is a simple service that allows you to upload an HTML file to use to send to prospective clients or for testing purposes.

Why does this service exist? There is no easy way to send HTML emails in most native e-mail clients. This makes things easier.

This site does not store any of your personal data and you must have an existing email service to send emails with. All data is cleared when you close your browser. All code is open source and available to anyone who is interested.

If you are using a Gmail account you will have to enable 3rd Party Services. Go to this article for instructions. Remember, you can only send up to 500 emails per day from gmail.
